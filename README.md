# Conversor de Temperatura em C

Aplicação desenvolvida em linguagem C para conversão de temperaturas entre Celsius e Fahrenheit.

## Funcionalidades
- Conversão de Celsius para Fahrenheit
- Conversão de Fahrenheit para Celsius
- Entrada de dados pelo usuário

## Tecnologias
- Linguagem C

## Como executar

Compile:
gcc conversor.c -o conversor

Execute:
./conversor

## Objetivo
Praticar lógica de programação, entrada de dados e estruturas condicionais.

- Menu interativo para escolha da conversão
